### Homebrew

1. Open Terminal
2. /bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"

1. brew tap mongodb/brew
2. brew install mongodb-community@4.4
3. brew services start mongodb-community@4.4
